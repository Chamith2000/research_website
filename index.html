AI-Powered Personalized Mathematics Learning Platform for Elementary Students

This AI-powered platform helps students learn mathematics in a smarter and more personalized way. It adjusts question difficulty, generates new startup questions, tracks progress, and provides learning recommendations based on each student's performance.

Students can track their progress, receive tailored learning resources, and identify their strengths and weaknesses over time. Teachers can monitor student performance, improve content using AI-powered feedback analysis, and ensure a fair learning experience with engagement and integrity monitoring.

Key Features
  • Adaptive Question Difficulty & MCQ Generation
  • Progress Tracking & Predictions
  • Content Quality Analysis
  • Engagement & Integrity Monitoring
Research Domain

Personalized Mathematics Learning for Elementary Students

This research presents an integrated AI-powered personalized mathematics learning platform for Grade 3, 4, and 5 students in Sri Lanka.

01

Literature Survey

Adaptive learning has been widely used to personalize education based on student performance. Traditional systems mainly rely on final marks, which do not fully capture student understanding. Modern approaches use Machine Learning and Learning Analytics to analyze additional factors such as response time, attempts, and engagement.

Deep learning models like LSTM are used to monitor and predict student progress over time, enabling early identification of learning difficulties. In addition, Large Language Models (LLMs) allow dynamic generation of questions instead of relying on fixed question banks, improving practice variety.

Recent research also highlights the importance of content quality and feedback analysis using NLP techniques, which help identify weaknesses in educational materials. Furthermore, emotion recognition and computer vision techniques are used to measure student engagement and ensure academic integrity in online learning environments.

However, most existing systems focus on only one or two of these aspects. Therefore, this project proposes an integrated AI-powered learning platform that combines adaptive learning, progress monitoring, content analysis, and engagement-based monitoring to provide more effective and personalized learning support.

02

Research Gap

Despite advancements, several limitations exist in current systems:

  • Most systems rely only on correctness or final marks
  • Fixed question banks limit practice variety
  • Lack of future performance prediction
  • Feedback is collected but not properly analyzed
  • Limited support for engagement monitoring and academic integrity

Therefore, there is a need for an integrated system that combines personalization, prediction, content analysis, and monitoring.

03

Research Problem

The main research problem is: How to develop a system that transforms student performance, behavior, and content data into personalized, reliable, and actionable learning support.

Sub-problems include:

  • Determining appropriate difficulty levels for students
  • Generating suitable questions dynamically
  • Monitoring and predicting student progress
  • Evaluating content quality and feedback
  • Ensuring engagement and preventing cheating in remote learning environments
04

Objectives

Main Objective

To design and develop an AI-powered personalized mathematics learning platform for elementary students.

Specific Objectives

  • Predict recommended difficulty level and number of questions
  • Generate structured MCQs using AI models
  • Monitor and forecast student learning progress
  • Analyze content quality and student feedback
  • Implement emotion-aware engagement and integrity monitoring
  • Integrate all components into a single system
05

Methodology

The project follows a modular AI-based system design built around four connected components.

Core Components

Adaptive Difficulty & Dynamic Question Generation

Recommends suitable difficulty levels and generates topic-based MCQs according to each learner's performance.

Continuous Progress Monitoring & Future Forecasting

Analyzes learner history to forecast future performance trends and support early intervention.

Content Quality & Feedback Analysis

Evaluates questions, teacher guides, and student feedback to identify content issues and improvement areas.

Emotion & Integrity Monitoring

Detects engagement signals and academic integrity concerns during remote learning sessions.

System Architecture
  • Frontend (user interaction)
  • Backend (application services)
  • AI/ML inference layer
  • Database layer

Frontend

JavaScript

Backend

Node.js

Runs backend services and API logic.

Express

Machine Learning / AI

Adaptive & Generation

Multi-task Neural Network

Predicts difficulty level and question count together.

Qwen2.5-Math-1.5B

Generates structured math-focused MCQs.

QLoRA

Fine-tunes the LLM with low memory cost.

Progress Monitoring

LSTM

Forecasts student progress over time.

ARIMA

Provides a statistical baseline for comparison.

Content Analysis

Qwen-based LLMs

Analyze content quality and feedback text.

LoRA

Fine-tunes content-analysis models efficiently.

Prompt Engineering

Guides consistent and relevant AI outputs.

Computer Vision

MobileNetV2

Recognizes learner emotion from facial cues.

YOLOv8

Detects objects and multiple humans for monitoring.

TensorFlow.js

Runs monitoring models directly in the browser.

Database

MongoDB

Stores profiles, attempts, logs, and analytics data.

ML / API Services

Flask

Exposes model inference through lightweight APIs.

Data Handling

JSON

Transfers structured data between components.

CSV

Stores and exports tabular datasets.
